# To-Do List JS

Aplicação web de lista de tarefas com interface moderna, armazenamento local e filtro de tarefas.

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Editar tarefa
- Remover tarefa
- Filtrar por status
- Persistência com localStorage

## Como executar
Basta abrir o arquivo `index.html` no navegador.
